# Fonomed Project

Fonomed is a healthcare management system designed for patients, doctors, and administrators.
The platform simplifies medical appointments, payment processes, and medical record management.

## Project Structure

- **backend/**: Handles the server-side logic, including APIs, authentication, and database integration.
- **frontend/**: Mobile application developed in React Native for patients and doctors.
- **Docker and Kubernetes**: Used for containerization and orchestration of services.
